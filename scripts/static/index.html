PACIO Sample Data

This page provides sample FHIR data for the PACIO Project. The PACIO (Post-Acute Care InterOperability) Project is a collaborative effort to advance interoperable health data exchange for post-acute care (PAC) patients. You can find more information at the main PACIO Project page.

The goal of this site is to publish sample data for interoperability testing, particularly for FHIR Connectathons, in a way that is accessible to both humans and machines. All sample data is generated from FHIR Shorthand (FSH) source files, which are managed in our public GitHub repository: paciowg/sample-data-fsh. The data is organized into the following sections:

  • Primary Releases are stable, versioned sets of data that correspond to official tags in the repository.
  • Working Releases are generated from the latest commits on active development branches and represent in-progress work.

For programmatic access, a manifest.json file is provided. This file contains a complete catalog of all sample data, organized by release and clinical scene, allowing automated tools to discover and retrieve specific data sets. For example, the PACIO pseudo-ehr reference implementation uses this to support loading of PACIO sample data onto FHIR servers.
